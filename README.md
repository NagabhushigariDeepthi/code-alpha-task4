# 🏨 Hotel Reservation System

## 📌 Overview

The **Hotel Reservation System** is a Java-based console application that allows users to search for available rooms, make and cancel reservations, simulate payments, and view booking details. The project is built using **Object-Oriented Programming (OOP)** principles and **File I/O** for persistent storage.

## ✨ Features

- Search available hotel rooms
- Room categorization (Standard, Deluxe, Suite)
- Book and cancel reservations
- Payment simulation
- View booking details
- Store booking records using File I/O (`bookings.txt`)

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Collections (`ArrayList`)
- File Handling (File I/O)
- Exception Handling

## ▶️ How to Run

1. Compile the program:
   ```bash
   javac HotelReservationSystem.java
   ```

2. Run the application:
   ```bash
   java HotelReservationSystem
   ```
